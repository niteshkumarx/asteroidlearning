C Language

The C is a general-purpose, procedural, imperative computer programming language developed in 1972 to develop the UNIX operating system. C is the most widely used computer language, it keeps fluctuating at number one scale of popularity along with Java programming language, which is also equally popular and most widely used among modern software programmers. You can program almost any thing out of C, its like big boss of every programming language. And still after more than four decades its still in use and dominating in system programming.

As in automation C is very helpful to program micro controllers and make it functional in no time. You can make LED patterns, make any electronic component function as per your requirement and time. Once you will see C to control the world you will fall in love with the language. Most of the operating systems and major applications are originally written in C and when it comes to speed of execution nothing beats C.
